# MongoDB Query Practice

A collection of MongoDB practice queries covering CRUD operations, comparison operators, logical operators, update operations, delete operations, document counting, and cursor methods.

## Topics Covered

### Comparison Operators
- `$gt` (Greater Than)
- `$lt` (Less Than)
- `$eq` (Equal To)
- `$ne` (Not Equal To)
- `$gte` (Greater Than or Equal To)
- `$lte` (Less Than or Equal To)

### Logical Operators
- `$and`
- `$or`

### CRUD Operations

#### Read Operations
- Find documents based on conditions
- Filter by category, price, and stock

#### Update Operations
- Update a single document
- Update multiple documents

#### Delete Operations
- Delete a single document
- Delete multiple documents

### Count Documents
- Count total documents
- Count documents matching conditions

### Cursor Methods
- Sort results
- Limit query output

## Sample Collection

The queries are based on a sample `products` collection containing fields such as:

```javascript
{
  name: "Laptop",
  category: "Electronics",
  price: 699,
  stock: 20
}
```
